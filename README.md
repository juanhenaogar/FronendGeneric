# InvestigacionFront

Front-end en **Blazor Web App (.NET 9)** para el módulo de investigación. Incluye CRUD de grupos de investigación, semilleros, participación de docentes y relaciones de líneas.

## Ejecución

```bash
dotnet restore
dotnet run
```

> No requiere backend. Los datos se manejan en memoria (`Services/InvestigacionDataService.cs`).
